## About The Project
  Console Application which help to do your daily work routine.

Why Amigo:
* It can search on wikipedia.
* It can open YouTube, Spotify, Whatsapp (if installed on your pc) and other cool stuff.
* You can easily add your command.


### Built With

* Python 3
